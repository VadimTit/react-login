Для создания нового приложения на React необходимо написать команды:
npx create-react-app my-app
cd my-app. 

Для запуска нам нужна команда npm start.
И если нету node_modules, необходимо их установить с помощью npm i

Я создал папку compopnents в которую поместил файл LoginForm.js. 
Экспортировал его в основной файл App.js.

У нас в форме авторизации есть данные админа, если мы их вводим правильно - то вход выполнен успешно, если нет - выскакивает ошибка: не правильный пароль или логин.

Можно добавить других пользователей и так же сделать проверку есть ли они в базе, а данные хранить в формате JSON .
